Maya Rigging Fundamentals

1. Foot Control

In this tutorial, I learned how to create foot controllers. This was done by creating a NURBS circle, snapping its pivot to the joint on the foot. Freezing the transformation and deleting the history. Then grouping it and rotating the group instead of the circle itself to preserve the value of tranforms of the circle to 0 while still rotating it. The NURBS circle was then modified to look like the outline of the foot only a bit bigger. This was duplicated to create the controller for the other foot. Now, when the controller is selected, it moves and pivots the leg.

2. Jaw Control

This tutorial started by using NURBS circle to create a controller for the jaw. Freezing its transforms and deleting its history. Then the pivot of the controller was snapped in to the jaw joint. The jaw drags when the neck is moving, and makes it look like the jaw is opened when the neck is moving even though it should not. This was solved by grouping the jaw controller and moving the groups pivot to the jaw joint and adding a parent constraint.

3. Stretchy Spine

If the spine is not set up correctly, when streching the spine, only the joint closest to the stretch is being stretched and not stretch the whole spine equally. This started by using the MEL editor and use the the arclen -ch1 function while the spine curve is selected to find out the lenght of the spine. Then the expression editor is used to start scripting. The script included $arclen = curveInfo1.arcLength/3.927; the dollar sign ($) is used to assign a new variable. The 3.927 came from the result of the MEL editor. Then CYC_Spine_JNT_01.scaleX = $arclen; . And this was repeated for the rest of the joints.
